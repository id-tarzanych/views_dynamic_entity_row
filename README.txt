## Overview

Views Dynamic Entity Row module provides dynamic row plugin that allows
to select individual view mode for each entity rendered by Views.

## How to use

1. Install Views Dynamic Entity Row module in preferred way.
2. Go to module's configuration page.
3. Select entity types and bundles that you want to support dynamic view mode.
4. Set view mode for entities on their add/edit form.
5. Create or edit View and select "<Entity Type> (dynamic)" row plugin
   in "Show" section.
